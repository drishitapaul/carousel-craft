## 🌐 Live Project

🔗 [View Published Project (Claude)]([https://carousel-craft-sandy.vercel.app])

---

## 🧠 Overview

CarouselCraft is a web-based tool that generates structured, high-quality social media carousel content from a single prompt.

It helps creators, educators, and marketers quickly turn ideas into:

* Multi-slide carousels
* Platform-optimized content
* Visually structured storytelling

---

## ✨ Features

* 🎯 Prompt → Full carousel generation
* 🧩 Structured slides (Hook → Build → Takeaway)
* ✏️ Editable slide content
* 🎨 Multiple formats (Carousel, Post, Story)
* 🌈 Theme customization
* 📥 Download slides as images
* 🧠 AI-powered content generation (Claude-based structure)

---

## 🛠️ Tech Stack

* React (Vite)
* TypeScript (TSX)
* Canvas API (for visuals)
* HTML2Canvas (for export)
* Claude API (design + generation logic)

---

## ⚙️ How It Works

1. User enters a content idea or selects a predefined pillar
2. AI generates structured JSON slides
3. Slides are rendered dynamically
4. User can edit, preview, and download

---

## 📌 Note

The project uses a Claude-based API structure.
For demo purposes, a mock system can be used if API access is unavailable.
